1.0 - Introdução

                As vezes precisamos mostrar aos nossos clientes, nossos produtos. Este software veio com o objetivo de facilitar isso, tanto pro cliente quanto pro dono do estabelecimento. Por meio de um painel de controle, você poderá colocar seus produtos em sua página, adicionando imagens, descrição, preço entre outras propriedades do produto que você desejar, e o melhor: Sem complicações.

2.0 - Dados Técnicos

                Esta seção é para dados técnicos, se você não possui conhecimentos sobre programação de computadores, banco de dados entre outras nomenclaturas técnicas, pode pular essa parte sem problemas.

                Este software é desenvolvido com PHP e MYSQL, com front-end feito em Twitter Bootstrap usando um layout Open Source chamado AdminLTE, ele requer um servidor que rode PHP > 5.6 e Mysql x.x. Ele foi baseado no modelo MVC e tem como seus arquivos principais, do CORE, na pasta Model.

3.0 - Instalação

        A instalação e implantação do sistema é simples, ele virá em um arquivo .ZIP com todas suas pastas e arquivos importantes e depois disso o único trabalho que terá é importar o banco para o Mysql e configurar os dados do mesmo no arquivo Banco.php que fica localizado na pasta Model dentro do local que está instalado o CataLOG.

3.2 - Banco de dados

        Nesta seção, falaremos sobre o banco de dados do sistema, como importá-lo e manuseá-lo

3.2.1 - Importando banco de dados

        

Esta parte é uma parte de suma importância para que o CataLOG funcione. Aqui fazemos a importação dos códigos do banco de dados para que possamos salvar as informações do CataLOG, do contrário, o sistema não funcionará.

OBS: Utilizaremos o PHPMyAdmin para fazermos a importação dos códigos, mas fique a vontade para utilizar qualquer meio de importação.

Dentro da pasta raíz dos arquivos do CataLOG, temos um arquivo que se chama , é este arquivo que vamos importar para o banco. Trata-se de um arquivo .sql que tem todas as informações do banco de dados.

  • O primeiro passo é abrir o PHPMyAdmin,após aberto, vamos ter algo como esta tela.

  • O próximo passo é clicar em importar, ou se sua versão do PHPMyAdmin for em inglês, “import”.
  • Depois de clicar vamos ter uma tela como essa.

  • Depois clicamos em escolher o arquivo, selecionamos o bd_catalogo.sql e clicamos em executar, depois de alguns instantes, teremos a base de dados funcionando.

  • Em seguida, vamos localizar as linhas que colocaremos as informações, como na imagem abaixo.

  • Essas linhas ficam entre a linha 140 e 160, São essas propriedades.

4.1.3 - Cadastro de categoria

        Este sistema possui um esquema de cadastro de categoria que é o seguinte: Como ele abrange todo tipo de produto, e determinados produtos possui alguns atributos específicos como por exemplo:

Os atributos de um produto da categoria automóveis não são os mesmos de uma categoria alimentícios, salvando preço, nome, descrição e imagem.

O cadastro de categoria veio para facilitar isso, pois você pode colocar atributos específicos àquela categoria.

A tela de cadastro de categoria é algo como isso.

Campo

Tipo

Existem dois campos de texto na tela, o campo Nome e o campo atributos específicos.

Este campo atributos específico é onde a mágica acontece, é aqui que você colocará o nome dos campos específicos que você queira naquela determinada categoria.

        Neste exemplo, eu vou demonstrar como cadastrar uma categoria Alimentícios que por sua vez possui atributos específicos como Lote, Data de fabricação e data de vencimento.

Como mostra na imagem, vamos preencher o campo nome.

Em seguida, temos que colocar nossos atributos específicos, que no caso são: Lote, Data de fabricação e data de vencimento.

Detalhe importante, os atributos são separados por vírgula “,”. Assim que pressionar a vírgula, vai surgir uma caixinha embaixo para que você possa marcar o tipo do campo que seu atributo vai ser, se é texto, data, etc… (tipos dos campos podem mudar de acordo com sua versão do CataLOG).

Após colocar os atributos separados por vírgula, teremos algo como isto.

Em seguida, vamos marcar quais serão os tipos dos campos dos nossos atributos específicos.

Que no nosso caso, serão texto, data e data, respectivamente.

Depois de marcados, teremos algo como isto.

Após marcados, é só clicar em cadastrar e nossa categoria vai estar pronta para uso!
